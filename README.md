# DeepSecure-AI-Cyber-Threat-Detection
AI-powered intrusion detection system using machine learning
DeepSecure is a machine learning-based intrusion detection system designed to classify network traffic as normal or malicious. The system uses advanced algorithms to detect cyber threats efficiently.

**Technologies Used**
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Seaborn

**Machine Learning Models**
Logistic Regression
Decision Tree
Random Forest

**Features**
Data preprocessing and cleaning
Feature scaling
Model training and evaluation
Confusion matrix visualization
Feature importance analysis

**Results**
Random Forest achieved the highest accuracy among all models, making it the best choice for intrusion detection.
